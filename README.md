# Game Design Document Creater

> Claude Code Skill - 游戏策划案生成器

## 简介

Game Design Document Creater 是一个 Claude Code Skill，通过交互式问答帮助用户生成完整的游戏设计文档(GDD)。

## 功能特性

### 🎮 快速模式
- 用户说"按照XX游戏设计"时自动分析参考游戏
- 展示分析结果供用户确认或修改
- 快速生成策划案

### 📝 新建策划案
- 系统性问答收集游戏信息
- 支持游戏类型细化（射击→FPS/TPS，RPG→ARPG/JRPG等）
- 自动生成结构化GDD文档

### ✏️ 优化已有策划案
- 自动检测项目内已有策划案
- 选择性优化特定模块
- 支持批量优化多个策划案

### 🔧 智能选项
- 使用选项框交互，体验友好
- 支持多选
- 每组选项都有"跳过"选项

## 使用方法

### 触发方式
- 输入 `/game-design-document-creater` 命令
- 说"帮我设计一个游戏"
- 说"生成游戏策划案"
- 说"按照XX游戏给我设计一个游戏"

### 安装
将 `SKILL.md` 文件复制到你的 Claude Code skills 目录：
```
~/.claude/skills/game-design-document-creater/SKILL.md
```

## 生成的文档结构

生成的 GDD 包含以下章节：
1. 游戏概述
2. 核心玩法设计
3. 系统详细设计
4. 数值设计框架
5. 美术需求清单
6. 技术架构建议
7. 开发里程碑
8. 风险评估
9. 版本历史

## 示例

```
用户: 按照原神给我设计一个游戏

助手: 📊 已分析《原神》的核心信息：

| 项目 | 分析结果 |
|------|----------|
| **游戏类型** | 开放世界动作角色扮演 |
| **美术风格** | 3D二次元动漫风 |
| **核心循环** | 探索收集 + 战斗刷宝 + 剧情体验 |
...

请问您是否需要修改这些元素？
1. ✅ 直接生成策划案
2. ✏️ 修改部分元素
```
