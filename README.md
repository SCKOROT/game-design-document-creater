# Game Design Document Creator

> Claude Code Skill - 游戏策划案生成器

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 📖 简介

Game Design Document Creator 是一个 Claude Code Skill，通过智能交互式问答帮助用户生成完整的游戏设计文档 (GDD)。

只需告诉 Claude 你想做什么游戏，它会自动引导你完成整个策划流程。

## ✨ 功能特性

### 🚀 快速模式
说"按照原神给我设计一个游戏"，自动分析参考游戏并生成策划案：
- 自动识别游戏类型、美术风格、核心循环
- 展示分析结果供确认
- 支持选择性修改后生成

### 📝 新建模式
完整的交互式问答流程：
- 游戏类型细化（射击→FPS/TPS，RPG→ARPG/JRPG等）
- 使用选项框友好交互
- 支持多选和跳过

### ✏️ 优化模式
优化已有策划案：
- 自动检测项目内策划案
- 选择性优化特定模块
- 支持批量优化多个策划案

## 🎮 使用方法

### 触发方式

| 方式 | 示例 |
|------|------|
| 命令 | `/gddcreator` |
| 自然语言 | "帮我设计一个游戏" |
| 参考模式 | "按照原神给我设计一个游戏" |
| 优化模式 | "帮我优化策划案" |

### 触发逻辑

```
用户输入 "按照原神设计游戏" ──→ ⚡ 快速模式
                    │
用户输入 "帮我设计游戏" ──→ 📝 新建模式
                    │
用户输入 "优化策划案" ──→ ✏️ 优化模式
                    │
命令 /gddcreator ──→ 有策划案？──→ 询问选择
                              └──→ 无策划案？──→ 📝 新建模式
```

## 📥 安装

将 `SKILL.md` 文件复制到 Claude Code skills 目录：

```bash
mkdir -p ~/.claude/skills/game-design-document-creater
cp SKILL.md ~/.claude/skills/game-design-document-creater/
```

├── 三、系统详细设计

│   └── 核心技术点
│
├── 七、开发里程碑
│   ├── Phase 1: 核心玩法验证

## 🎬 使用示例

### 示例1：快速模式

```
用户: 按照原神给我设计一个游戏

助手: 📊 已分析《原神》的核心信息：

| 项目 | 分析结果 |
|------|----------|
| **游戏类型** | 开放世界动作角色扮演 |
| **美术风格** | 3D二次元动漫风 |
| **核心循环** | 探索收集 + 战斗刷宝 + 剧情体验 |
| **目标平台** | 多平台 |
| **商业模式** | 免费内购 + 抽卡系统 |
| **核心特色** | 元素反应战斗 + 开放世界探索 |

请问您是否需要修改这些元素？
1. ✅ 直接生成策划案
2. ✏️ 修改部分元素
```

### 示例2：新建模式

```
用户: 帮我设计一个麻将游戏

助手: 问题 1/9: 你想制作什么类型的游戏？

- 动作游戏 (ACT)
- 角色扮演 (RPG)
- 策略游戏 (SLG/RTS)
- 休闲游戏
- 卡牌/回合制
- ...

用户: 休闲游戏

助手: 问题 1.1: 休闲游戏的具体类型是？

- 消除类
- 麻将/棋牌类
- 跑酷类
- 益智解谜
- ...

用户: 麻将/棋牌类

助手: 问题 1.2: 麻将游戏的具体玩法是？

- 经典麻将
- 四川血战麻将
- 国标麻将
- 麻将竞技对战
- 麻将+RPG
- ...
```

### 示例3：优化模式

```
用户: 帮我优化策划案

助手: 请选择需要优化的策划案：

| 编号 | 文件名 | 游戏名称 |
|------|--------|----------|
| 1 | Docs/GDD.md | 《战锤40K：异形猎手》 |
| 2 | Docs/GDD2.md | 《元素幻想》 |

用户: 1

助手: 您希望优化哪些方面？

1. 🎮 核心玩法
2. 📊 数值系统
3. 🎨 美术风格
4. 💰 商业模式
...
```

## 📊 支持的游戏类型

| 大类 | 细分类型 |
|------|----------|
| 射击游戏 | FPS、TPS、俯视角、横版、混合 |
| RPG | ARPG、JRPG、MMORPG、挂机RPG、Roguelike |
| 策略游戏 | SLG、RTS、塔防、战棋 |
| 动作游戏 | 砍杀类、平台跳跃、格斗、动作冒险 |
| 休闲游戏 | 消除、麻将/棋牌、跑酷、益智 |
| 卡牌/回合制 | 策略卡牌、回合战斗 |
| 模拟经营 | 经营管理、资源调配 |
